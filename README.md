# 小鹤音形-Rime

因为我习惯输入两个字的词，而小鹤音形原生方案鼓励单字输出，所以我用 Rime 维护了一份自定义词库。这个词库继承了小鹤音形绝大部分单字的输出方式和拼写方式，并加入了大量双字词拼写。双字词主要来自搜狗词库，也包含我长期使用中手动整理的常用词和自定义短语。

这份词库由我创建并维护，前后花了接近十年时间。在前AI时代我用爬虫和算法完成了雏形，直至今年在 Codex 的协助下完成了结构整理。如果你正好需要这个词库，希望你能点一个Star让我知道。

## 文件说明

`custom_phrase.txt` 是 Rime 的 `tabledb` 文本词库，基本格式为：

```text
词条	编码	权重
```

使用时，将 `custom_phrase.txt` 放入 Rime 用户目录，并在方案配置中启用 `table_translator@custom_phrase`。

7z文件是我的其他rime配置，用于个人备份。
