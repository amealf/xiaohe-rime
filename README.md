# 小鹤音形-Rime

因为我习惯输入两个字的词，而小鹤音形原生方案鼓励单字输出，所以我用 Rime 维护了一份自定义词库。这个词库继承了小鹤音形绝大部分单字的输出方式和拼写方式，并加入了大量双字词拼写。双字词主要来自搜狗词库，也包含我长期使用中手动整理的常用词和自定义短语。

这份词库由我创建并维护，前后花了数年时间。今年在 Codex 协助下完成了一次权重统一和结构整理，形成了符合我输入习惯的版本。公开文件已经移除个人自定义短语和账号相关内容。如果你正好需要这样的 Rime 词库，可以自取；如果它对你有帮助，希望点一个 Star，让我知道它真的帮到了别人。

## 文件说明

`custom_phrase.txt` 是 Rime 的 `tabledb` 文本词库，基本格式为：

```text
词条	编码	权重
```

当前文件已经统一第三列权重口径。同一个编码下，候选顺序延续公开源词条的顺序，权重只用于固定候选先后，避免一份词库里同时混用 `1`、`1000`、`10000`、`100000` 等不同口径。

这个词库适合使用小鹤音形输入方式、并且希望增加大量双字词输入的人。使用时，将 `custom_phrase.txt` 放入 Rime 用户目录，并在方案配置中启用 `table_translator@custom_phrase`。
